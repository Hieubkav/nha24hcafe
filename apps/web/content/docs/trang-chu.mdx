<Callout type="success" title="Mục tiêu">
Truyền tải nhanh USP (mở 24/7, phòng họp miễn phí, bàn làm việc riêng tư, sleep box, không gian 1.000m²), thúc đẩy hành động (xem menu/đặt chỗ), hiển thị hình ảnh đẹp và tối ưu SEO.
</Callout>

## Cấu trúc nội dung

- Hero: ảnh nền ấn tượng + tiêu đề + CTA.
- USP nhanh: 4–6 điểm mạnh (badge/icon).
- Rooms/Spaces: teaser Meeting room, Private desk, Sleep box.
- Menu nổi bật: 4–8 món signature (khi có dữ liệu).
- Gallery: 6–12 ảnh nội thất đẹp nhất.
- Reviews trích dẫn (nếu có).
- Visit us: giờ mở cửa, địa chỉ, chỉ đường (map), liên hệ.
- CTA cuối trang.

## Mapping dữ liệu → UI

| Phần | Nguồn dữ liệu | Ghi chú |
| --- | --- | --- |
| Hero | `images.role = "hero"` trong `data/nha24h.json` | fallback chọn ảnh gallery rộng |
| USP | `amenities[]` | map sang badge/text |
| Rooms/Spaces | `rooms[]` (khi có) | type: meeting/private/sleep_box |
| Menu nổi bật | `menuItems[]` (khi có) | theo tag `signature` hoặc curated |
| Gallery | `images.role = "gallery"` | giới hạn 6–12 ảnh |
| Reviews | `reviews[]` (khi có) | rating, comment, source |
| Visit us | `hours[]`, `address`, `contact` | hiển thị tóm tắt |

## Tiến độ hiện tại

- Đã xong: Hero, Features (giá/giờ/liên hệ), Gallery (tối đa 12 ảnh).
- Đang chờ dữ liệu: Menu, Rooms/Booking, Reviews.
- Kế hoạch: Thêm section Visit us (map + chỉ đường), CTA cuối trang.
