## Phân lớp

### Frontend

- Next.js App Router, React 19.
- Tailwind v4 + shadcn/ui (UI primitives cho form/table/dialog...).
- Fumadocs (docs nội bộ: MDX, Cards, Callout, CodeBlock, Table).

### Backend/Realtime (định hướng)

- Convex: schema, API, realtime cho Admin/booking/order.

### Dev & Build

- Monorepo Turbo.
- Môi trường web trong `apps/web`.
- Ảnh tĩnh trong `apps/web/public/nha24h`.

## Lệnh hữu ích

```bash
# Chạy web (turbopack)
bun run dev:web

# Build
bun run build

# Kiểm tra lint (root)
bun run check
```
